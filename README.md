# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

## Navigate to the project folder

```bash
cd portfolio
```

## Install dependencies

```bash
npm install
```

## Start development server

```bash
npm run dev
```
